How to use CarcassHub?

To use CarcassHub run the executable you get from unzipping the file through the Command Prompt, and choose the first option.

After it's done you can restart Command Prompt and use it with "CarcassI option(run/cmd)"

Currently, CarcassHub is a Windows-only item, so if you wanna use CarcassI, compile it from the source in the Github.

Will there be a Linux release for CarcassHub?

Yes, there will be. It's most likely coming out very soon, for now either compile CarcassI from source or try using Mono on it.
